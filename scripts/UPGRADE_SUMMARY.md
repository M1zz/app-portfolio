## 추가된 Feature의 구조

각 feature는 다음 7가지 정보를 포함합니다:

1. **category**: 사용자 중심 카테고리 (예: "빠른 입력", "건강 관리", "데이터 안전")
2. **description**: 무엇을 할 수 있는가
3. **userValue**: 왜 유용한가
4. **technicalNotes**: 기술 구현 방법
5. **usageScenario**: 언제 사용하는가
6. **problemSolved**: 어떤 문제를 해결하는가
7. **userBenefit**: 구체적으로 어떤 이득이 있는가
