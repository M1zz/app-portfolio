### 1. 대시보드 탭
현재 화면에서 확인 가능:
- 4개 KPI 요약 카드
- 우선순위 높은 앱 목록
- 진행률 도넛 차트
- 위험 요소 경고

### 2. 앱 그리드 탭
23개 앱을 카드 형태로 표시:
- 각 앱의 진행률
- 다음 태스크
- 상태 색상 표시

### 3. 브리핑 탭
CEO 일일 브리핑:
- "터미널에서 브리핑 생성" 버튼 클릭
- Claude가 자동으로 분석 및 생성
- 결과를 앱에서 바로 확인

### 4. 결정 센터 탭
의사결정 및 요청사항:
- 대기 중인 결정 표시
- 대기 중인 요청 표시
- "모든 결정/요청 처리" 버튼

### 5. 단축키
- **⌘K**: 앱 빠른 검색
- **⌘R**: 데이터 새로고침
