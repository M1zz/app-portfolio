## 버그 정보
- 증상: {버그 설명}
- 재현 단계: {재현 방법}
- 발생 환경: iOS {버전}, 앱 버전 {버전}
- 크래시 로그: {로그 첨부}

## 조사 절차
1. 관련 코드 위치 파악
   - 에러 스택 트레이스 분석
   - 해당 기능 코드 읽기

2. 유사 이슈 검색
   - 다른 앱에서 비슷한 버그 있었는지 확인
   - 과거 해결 방법 찾기

3. 원인 분석
   - 버그 발생 원인 설명
   - 재현 가능한 최소 코드 작성

4. 해결 방안 제시
   - 수정 방법 설명
   - 코드 변경사항 제안
   - 테스트 케이스 작성

5. 예방 조치
   - 비슷한 버그 방지 방법
   - 추가 테스트 필요 여부

단계별로 진행하면서 발견사항을 보고해줘.
