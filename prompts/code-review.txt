# 코드 리뷰 프롬프트
# 사용법: claude < prompts/code-review.txt

{앱이름} PR #{번호}를 리뷰해줘:

## 리뷰 체크리스트

### 1. 코드 품질
- [ ] 가독성 (명확한 변수명, 함수명)
- [ ] Swift 스타일 가이드 준수
- [ ] 중복 코드 제거
- [ ] 적절한 주석

### 2. 기능 완성도
- [ ] 요구사항 충족
- [ ] 엣지 케이스 처리
- [ ] 에러 핸들링

### 3. 성능
- [ ] 불필요한 연산 없음
- [ ] 메모리 누수 없음
- [ ] UI 블로킹 없음

### 4. 테스트
- [ ] 단위 테스트 작성
- [ ] 테스트 커버리지 적절
- [ ] 모든 테스트 통과

### 5. 보안
- [ ] 민감한 정보 하드코딩 없음
- [ ] 입력 검증
- [ ] 안전한 데이터 저장

### 6. 재사용성
- [ ] 다른 앱에서도 사용 가능한 코드인지
- [ ] 공유 모듈로 추출 가능한지

## 리뷰 결과
- ✅ 승인 / ⚠️ 수정 필요 / ❌ 재작업 필요
- 주요 피드백
- 개선 제안

객관적이고 건설적인 피드백을 해줘.
