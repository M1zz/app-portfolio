# 배포 체크리스트 프롬프트
# 사용법: claude < prompts/deploy-checklist.txt

{앱이름} v{버전} 배포를 준비해줘:

## 1단계: 사전 검증
- [ ] 모든 테스트 통과 확인
- [ ] 버전 번호 업데이트 (Info.plist, 앱 JSON)
- [ ] 변경사항 문서화

## 2단계: 릴리스 노트 생성
- [ ] 완료된 기능 정리
- [ ] 버그 수정 내역 정리
- [ ] 앱스토어용 설명 작성 (500자 이내)
- [ ] 영어 버전도 작성

## 3단계: 빌드 및 제출
- [ ] 릴리스 빌드 생성
- [ ] 스크린샷 업데이트 필요 여부 확인
- [ ] TestFlight 업로드
- [ ] 내부 테스터 배포

## 4단계: 포트폴리오 업데이트
- [ ] apps/{앱파일}.json에서 currentVersion 업데이트
- [ ] 완료된 태스크들을 done으로 변경
- [ ] portfolio-summary.json 재생성

## 5단계: 문서화
- [ ] CHANGELOG.md 업데이트
- [ ] 배포 완료 기록

각 단계별로 확인하면서 진행해줘. 문제가 있으면 즉시 알려줘.
