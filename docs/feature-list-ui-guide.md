# 피쳐리스트 UI/UX 구조화 가이드

> 223개 feature를 효과적으로 보여주는 정보 설계

## 📋 문제 정의

**현재 상황**
- 19개 앱 × 평균 11.7개 feature = 223개
- 각 feature마다 7가지 정보 (category, description, userValue, technicalNotes, usageScenario, problemSolved, userBenefit)
- 카테고리만 134개

**목표**
- 한눈에 파악 가능한 구조
- 원하는 정보를 빠르게 찾기
- 제품 로드맵으로 활용 가능

---

## 🎯 정보 계층 설계 (3단계)

### 1단계: Overview (전체 조감도)
```
앱 선택 바
   ↓
카테고리별 요약
   ↓
각 카테고리당 feature 개수 + 상태 분포
```

### 2단계: Feature Cards (카드 뷰)
```
카테고리 헤더
   ↓
Feature 카드 그리드
   - 이름 (Bold)
   - 한줄 요약 (usageScenario의 첫 문장)
   - 상태 배지 + 카테고리 태그
```

### 3단계: Feature Detail (상세 뷰)
```
Feature 클릭 시 Sheet 표시
   - 전체 7가지 정보
   - 관련 태스크
   - 타임라인
```

**구현 포인트**

## 🎯 정보 우선순위 설계

### 카드 표면 (항상 보임)
```swift
1. Feature 이름 (Bold, 18pt)
2. 카테고리 태그 (색상 구분)
3. 상태 배지 (✅ 🔄 💡)
4. 한줄 요약 (usageScenario 첫 문장)
```

### 호버/미리보기 (마우스 올리면)
```swift
5. userBenefit (강조)
6. problemSolved (보조)
```

### 상세 뷰 (클릭 시)
```swift
전체 7가지 정보
├─ 📌 usageScenario (언제 사용?)
├─ ❌ problemSolved (무엇 해결?)
├─ ✨ userBenefit (무엇 얻나?)
├─ 💡 description (무엇인가?)
├─ 🎯 userValue (왜 유용?)
└─ ⚙️ technicalNotes (어떻게?)
```

## 🎨 색상 코딩 시스템

### 카테고리별 색상
```
빠른 입력       → 파란색 계열
기기 간 연동    → 초록색 계열
데이터 안전     → 빨간색 계열
절약 시간 확인  → 주황색 계열
건강 관리       → 민트색 계열
감정 표현       → 보라색 계열
```

### 상태별 색상
```
✅ 완료 (done)          → 초록 배경
🔄 진행중 (in-progress) → 파란 배경
💡 계획 (todo)          → 노란 배경
⚪ 대기 (not-started)   → 회색 배경
```

## 🔍 검색 & 필터 설계

### 검색
```
[🔍 검색: "시간 절약"]
    ↓
결과:
  - 템플릿 기능 (userBenefit: "5-10분 절약")
  - 통계 대시보드 (category: "절약 시간 확인")
  - 위젯 지원 (usageScenario: "시간 낭비 없이")
```

### 다중 필터
```
앱:        [클립키보드] [세끼] [라포맵]
카테고리:  [빠른입력] [건강관리]
상태:      [완료] [진행중]
버전:      [v3.0] [v3.1]
```

## 📱 실제 UI 예시

### 카드 컴포넌트 구조
```swift
struct FeatureCard: View {
    var body: some View {
        VStack(alignment: .leading, spacing: 8) {
            // 상태 + 카테고리
            HStack {
                StatusBadge(status: "완료")
                Spacer()
                CategoryTag(category: "빠른 입력")
            }

            // Feature 이름
            Text("템플릿 기능 추가")
                .font(.headline)

            // 한줄 요약
            Text("업무 중 자주 쓰는 문구를 매번 타이핑하는 상황")
                .font(.caption)
                .foregroundColor(.secondary)
                .lineLimit(2)

            // 사용자 이득 (강조)
            Label("일일 5-10분 절약", systemImage: "star.fill")
                .font(.caption)
                .foregroundColor(.orange)
        }
        .padding()
        .background(Color.gray.opacity(0.1))
        .cornerRadius(12)
    }
}
```

```

⭐ 중간: 차별화 요소
💡 낮음: 보조 기능
```

### 3. 완성도 표시
```
Feature 카드에 진행률:
[████████░░] 80% 완성
```

```
"템플릿 기능" 클릭 시:
  → 관련: "카테고리 관리", "통계 기능"
```
