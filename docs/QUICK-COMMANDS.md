# ⚡ 빠른 명령어 가이드

## 📱 앱별 작업

### 상태 확인
```
"두 번 알림 상태 알려줘"
"라포 맵 진행률 확인해줘"
"세끼 다음 태스크 뭐야?"
```

### 기능 추가
```
"두 번 알림에 위젯 다크모드 추가해줘"
"라포 맵에 사진 필터 기능 넣어줘"
"세끼에 식단 기록 기능 만들어줘"
```

### 버그 수정
```
"클립키보드 복사 버그 고쳐줘"
"쿨타임 타이머 멈춤 현상 해결해줘"
```

### 결정 사항 기록
```
"두 번 알림 결정: SwiftData 마이그레이션"
"라포 맵 decisions-log에 추가: 클라우드 백업 연기"
```

## 📊 전체 관리

### 현황 파악
```
"전체 앱 현황 요약해줘"
"우선순위 높은 앱들 상태 알려줘"
"진행률 낮은 앱 찾아줘"
"완료 임박한 앱 알려줘"
```

### 비교 분석
```
"두 번 알림이랑 라포 맵 비교해줘"
"iOS 16+ 앱들 목록 알려줘"
"위젯 있는 앱들 보여줘"
```

### 계획 수립
```
"이번 주 작업할 앱 3개 추천해줘"
"내일 할 일 정리해줘"
"다음 릴리스 계획 세워줘"
```

## 🎯 자주 쓰는 패턴

### 작업 시작
```
"오늘 뭐부터 할까?"
"[앱이름] 작업 시작할게"
```

### 작업 전환
```
"[앱이름] 끝났어, 다음 뭐 할까?"
"이제 [앱이름] 볼게"
```

### 하루 정리
```
"오늘 작업 요약해줘"
"변경된 내용 정리해줘"
"내일 할 일 추천해줘"
```

## 💡 예시

### 아침
```
You: "오늘 작업할 앱 추천해줘"
Claude: [우선순위 + 진행 가능한 앱 제안]

You: "두 번 알림부터 시작할게. 상태 알려줘"
Claude: [컨텍스트 로드 → 현재 상태 제공]
```

### 작업 중
```
You: "두 번 알림에 다크모드 추가해줘"
Claude: [architecture.md, design-system.md 참조 → 구현]

You: "좋아, 이제 라포 맵 볼게"
Claude: [컨텍스트 전환 → 라포 맵 정보 제공]
```

### 저녁
```
You: "오늘 뭐 했는지 정리해줘"
Claude: [변경사항 요약]

You: "내일 할 일 추천해줘"
Claude: [진행 상황 기반 제안]
```
